Seletors CSS

No CSS nós possuimos os seletores por ID

  • Seletor por tipo: Busca elementos por uma tag HTML especificada
  • Seletor por id: Busca um elemento com o atributo "id" igual adeterminado valor
  • Seletor por class: Busca todos os elementos que tenham a class definida no seu atributo "class"
  • Seletor por attribute: Busca todos os elementos que tenham algum atributo com determinado valor
  • Seletor Universal: Busca todos os elementos da página
